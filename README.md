## Usage

### Step 1: Fork (requires login)

### Step 2: Make changes

### Step 3: Copy the **new** url
Eg. `https://codesandbox.io/p/sandbox/goofy-silence-9tvzgq`

**Make sure it's not the original!**
`https://codesandbox.io/p/sandbox/github/vueform/sandbox/tree/main`
